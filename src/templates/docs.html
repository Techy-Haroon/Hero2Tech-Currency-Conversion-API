Welcome to the API Documentation

This API documentation provides details on the endpoints available for converting currencies, retrieving currency rates, and more.

You can find documentation for the following endpoints:

  • /convert: Convert a specific amount between two currencies.
  • /convert_currencies: Convert a base currency to multiple target currencies.
  • /get_currency_rates: Retrieve the latest rates for all available currencies.

For additional information on response codes and using the API, please visit the Response Codes page.
