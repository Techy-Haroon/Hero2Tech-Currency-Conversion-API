API Documentation

Endpoint: /get_currency_rates

Description

The /get_currency_rates endpoint retrieves the current currency rates for all available currencies. This endpoint is useful for getting a snapshot of exchange rates at a given time.

Methods

GET

URL

/get_currency_rates

Request Headers

  • Authorization: Bearer <your_api_key> (required if sending Authorization Header)

Request Parameters

Parameters can be passed in the URL. If using the Authorization header, then this will be ignored.

  • api_key (string, required if Authorization header is absent): The API key associated with your account.

Response

  • 200 OK: Returns a JSON object containing the current rates for all currencies.
  • 401 Unauthorized: Returned if the API key is missing or invalid.
  • 403 Forbidden: Returned if the You have reached your Request Limit.

Response Body Example

{
  "USD": 1,
  "PKR": 278.63,
  "EUR": 0.84,
  "INR": 7380.00,
  150+ More Currency Rates...
}

Python Example (GET with Authorization Header)

import requests
{% if session["user_id"] %}
api_key = "{{ session['api_key'] }}"
{% else %}
api_key = "your_api_key"
{% endif %}
url = "https://www.hero2tech.com/get_currency_rates"
headers = {
    "Authorization": f"Bearer {api_key}"
}

response = requests.get(url, headers=headers)
if response.status_code == 200:
    print(response.json())
else:
    print(f"Error: {response.status_code}, {response.text}")
